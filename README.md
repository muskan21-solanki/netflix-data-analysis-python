
# Netflix Data Analysis Project

## Project Overview
This project performs Exploratory Data Analysis (EDA) on Netflix titles data using Python, Pandas, NumPy, Matplotlib, and Seaborn.

The analysis focuses on:
- Content trends over the years
- Movies vs TV Shows distribution
- Genre/category analysis
- Country-wise content production
- Ratings distribution
- Duration insights

## Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Dataset
Download the dataset from Kaggle:
https://www.kaggle.com/datasets/shivamb/netflix-shows

Place the CSV file inside the `data` folder.

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the project:
   python netflix_analysis.py

3. Generated charts will be saved inside the `outputs` folder.
